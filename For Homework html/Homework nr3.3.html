Description List

HTML:
Este o chestie care ne ajuta sa punem ceva informatie sau mai corect zis litere,imagini etc. unde noua imi trebuie si sa aranjam.
HTml element
element astai tot html dar cind in cod infara de tag mai este ceva numai tag astai nui element trebuie sa mai fie ceva
HTML TAG
tag astai este ceva ce ne ajuta sa punem sa marim sau sa coloram sau sai dam limba,titlu uncoded etc. cu tagul aratam ce vom folosi !
HTML ATTRIBUTE
ATTRIBUTE astai este o valoarea la tag sau nu la tag care ne ajuta , ceia ce nu vedem utilizatorul , sa caute sa coloreze sa puna limba , informatie in ATTRIBUTE poate sa fie diferita dar trebuie sa stii ce ATTRIBUTE trebuie sa folosesti!
HTML COMMENT
comment asta este o chestie ca ajutor cind tu vezi codul dupa cineva sau dupa ce ai fost in holiday ! informatie care ne ajuta sa intelegem pentru cei aceasta ! si care utilizatorul nu o vede numai programmistul o vede!
